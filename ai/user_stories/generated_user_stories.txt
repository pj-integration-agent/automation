User Stories:

US01 - Como um Especialista do Banco, eu quero criar um novo registro de Banco Nacional para que eu possa cadastrar as informações essenciais do banco.

Descrição:
- Na tela de cadastro de Banco Nacional, os seguintes campos devem ser preenchidos obrigatoriamente:
  - Código (que não pode ser igual a outro já cadastrado)
  - Descrição do Banco
  - Apelido
  - Número de inscrição no SBP
  - Banco controlador (lista suspensa)
  - CNPJ (deve ser válido e existente)
- Após o preenchimento de todos os campos obrigatórios, o usuário deve clicar no botão "Salvar" para validar o cadastro.

Critérios de Aceitação:
- O Código do Banco deve ser único, não podendo ser igual a outro já cadastrado.
- Todos os campos obrigatórios devem ser preenchidos para que o cadastro seja realizado.
- O CNPJ informado deve ser válido e existente.
- Após clicar em "Salvar", o cadastro deve ser validado e o novo registro deve ser exibido na listagem de Bancos Nacionais.

Rastreabilidade: US01

US02 - Como um Especialista do Banco, eu quero editar as informações de um Banco Nacional cadastrado para que eu possa manter os dados atualizados.

Descrição:
- Na tela de edição de Banco Nacional, os seguintes campos poderão ser editados:
  - Descrição do Banco
  - Apelido
  - Número de inscrição no SBP
  - Banco controlador (lista suspensa)
  - CNPJ
- O campo "Código" não poderá ser editado.
- Após realizar as alterações desejadas, o usuário deve clicar no botão "Salvar" para validar as modificações.

Critérios de Aceitação:
- O Código do Banco não poderá ser editado.
- Todos os outros campos permitidos para edição devem ser atualizados corretamente.
- O CNPJ informado deve ser válido e existente.
- Após clicar em "Salvar", as alterações devem ser validadas e o registro atualizado deve ser exibido na listagem de Bancos Nacionais.

Rastreabilidade: US02
