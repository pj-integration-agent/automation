User Stories:

US01 - Cadastrar novo banco nacional
Como um QA especialista, eu quero poder cadastrar um novo banco nacional para que a aplicação contenha todas as informações necessárias desse banco.

Descrição:
Na tela de cadastro de novo banco, devo preencher os seguintes campos obrigatórios:
- Código (não pode ser igual a outro já cadastrado)
- Descrição do Banco
- Apelido
- Número de inscrição no SBP
- Banco controlador (em uma lista)
- CNPJ (deve ser válido e existente)

Após o preenchimento de todos os campos obrigatórios, devo clicar no botão "Salvar" para validar o cadastro do novo banco.

Critérios de Aceitação:
- Todos os campos obrigatórios devem ser preenchidos
- O código não pode ser igual a outro já cadastrado
- O CNPJ deve ser válido e existente
- Ao clicar em "Salvar", o cadastro deve ser concluído com sucesso

Rastreabilidade: US01

US02 - Editar banco nacional existente
Como um QA especialista, eu quero poder editar as informações de um banco nacional já cadastrado para manter os dados atualizados.

Descrição:
Ao selecionar um banco existente, devo poder editar os seguintes campos:
- Descrição do Banco
- Apelido
- Número de inscrição no SBP
- Banco controlador
- CNPJ

O campo "Código" não deve ser editável, pois é um identificador único do banco.

Após realizar as alterações desejadas, devo clicar no botão "Salvar" para validar as modificações.

Critérios de Aceitação:
- Os campos editáveis devem ser atualizados corretamente
- O campo "Código" não deve ser editável
- Ao clicar em "Salvar", as alterações devem ser salvas com sucesso

Rastreabilidade: US02

BDD - Cenários Positivos:

Cenário: Cadastrar novo banco nacional com sucesso
Dado que eu esteja na tela de cadastro de novo banco
Quando eu preencher todos os campos obrigatórios corretamente
E clicar no botão "Salvar"
Então o cadastro do novo banco deve ser concluído com sucesso

Cenário: Editar informações de um banco nacional existente
Dado que eu tenha um banco nacional já cadastrado
E eu esteja visualizando as informações desse banco
Quando eu alterar os campos editáveis
E clicar no botão "Salvar"
Então as alterações devem ser salvas com sucesso

BDD - Cenários Negativos:

Cenário: Tentar cadastrar novo banco com código já existente
Dado que eu esteja na tela de cadastro de novo banco
Quando eu preencher todos os campos corretamente
E informar um código que já existe em outro registro
E clicar no botão "Salvar"
Então o sistema deve exibir uma mensagem de erro informando que o código já existe

Cenário: Tentar cadastrar novo banco com CNPJ inválido
Dado que eu esteja na tela de cadastro de novo banco
Quando eu preencher todos os campos corretamente
E informar um CNPJ inválido
E clicar no botão "Salvar"
Então o sistema deve exibir uma mensagem de erro informando que o CNPJ é inválido

Cenário: Tentar editar código de um banco existente
Dado que eu esteja visualizando as informações de um banco nacional cadastrado
Quando eu tentar alterar o campo "Código"
Então o sistema deve bloquear a edição desse campo
