**User Story 1 – Criação de um novo banco (US01)**  

```yaml
titulo: Criação de um novo banco (US01)
cenario_bdd:
    nome: Abrir tela de cadastro ao clicar em Novo
    tipo: positivo
    gherkin: "Feature: Cadastro de bancos\n\
Scenario: Abrir tela de cadastro ao clicar em Novo\n\
  Given eu estou na lista de bancos\n\
  When eu clico em 'Novo'\n\
  Then a tela de cadastro de bancos deve ser exibida\n\
  And os campos obrigatórios são marcados com '*'\n"
```

```yaml
titulo: Criação de um novo banco (US01)
cenario_bdd:
    nome: Registrar banco com todos os campos válidos
    tipo: positivo
    gherkin: "Feature: Cadastro de bancos\n\
Scenario: Registrar banco com todos os campos válidos\n\
  Given eu estou na tela de cadastro de bancos\n\
  When eu preencho o campo 'Código' com 'BAN001'\n\
  And eu preencho o campo 'Descrição do Banco' com 'Banco do Norte'\n\
  And eu preencho o campo 'Apelido' com 'Norte'\n\
  And eu preencho o campo 'Número de inscrição no SBP' com '12345678'\n\
  And eu seleciono o campo 'Banco Controlador' com o valor 'Banco Central'\n\
  And eu preencho o campo 'CNPJ' com '12.345.678/0001-95'\n\
  And eu clico em 'Salvar'\n\
  Then devo ver a mensagem 'Banco cadastrado com sucesso.'\n\
  And o banco com código 'BAN001' deve aparecer na lista de bancos\n"
```

**User Story 2 – Edição de um banco existente (US02)**  

```yaml
titulo: Edição de um banco existente (US02)
cenario_bdd:
    nome: Abrir tela de edição e verificar campo Código somente leitura
    tipo: positivo
    gherkin: "Feature: Edição de bancos\n\
Scenario: Abrir tela de edição e verificar campo Código somente leitura\n\
  Given eu estou na lista de bancos\n\
  When eu seleciono o banco com código 'BAN001' e clico em 'Editar'\n\
  Then a tela de cadastro deve abrir com os dados preenchidos\n\
  And o campo 'Código' deve ser somente leitura\n"
```

```yaml
titulo: Edição de um banco existente (US02)
cenario_bdd:
    nome: Editar banco mantendo o código inalterado
    tipo: positivo
    gherkin: "Feature: Edição de bancos\n\
Scenario: Editar banco mantendo o código inalterado\n\
  Given eu estou na tela de cadastro do banco 'BAN001'\n\
  When eu altero o campo 'Descrição do Banco' para 'Banco do Sul'\n\
  And eu altero o campo 'Apelido' para 'Sul'\n\
  And eu altero o campo 'Número de inscrição no SBP' para '87654321'\n\
  And eu altero o campo 'Banco Controlador' para 'Banco Regional'\n\
  And eu altero o campo 'CNPJ' para '98.765.432/0001-10'\n\
  And eu clico em 'Salvar'\n\
  Then devo ver a mensagem 'Banco atualizado com sucesso.'\n\
  And o banco na lista deve manter o mesmo Código 'BAN001'\n"
```

