**US02 – Edição de Banco Existente**  
```
titulo: Edição de banco existente
cenario_bdd:
    nome: Edição bem‑sucesso de campos permitidos
    tipo: positivo
    gherkin: "Feature: Edição de banco existente\n  Scenario: Edição bem-sucedida de campos permitidos\n    Given que existe um banco cadastrado com código 'BAN002'\n    When abro o registro 'BAN002' para edição\n    And altero o campo 'Descrição do Banco' para 'Banco do Sul'\n    And altero o campo 'Apelido' para 'BS'\n    And altero o número de inscrição no SBP para '654321'\n    And altero o Banco Controlador para 'Banco Central'\n    And altero o CNPJ para '98.765.432/0001-10'\n    When clico no botão 'Salvar'\n    Then o registro 'BAN002' é atualizado na lista\n    And as alterações refletem corretamente"

cenario_bdd:
    nome: Tentativa de alteração do Código
    tipo: negativo
    gherkin: "Feature: Edição de banco existente\n  Scenario: Tentativa de alteração do Código\n    Given que existe um banco cadastrado com código 'BAN003'\n    When abro o registro 'BAN003' para edição\n    And tento alterar o campo 'Código' para 'BAN004'\n    Then o campo permanece somente‑leitura e não há alteração\n    And o botão 'Salvar' salva apenas os campos modificados"

cenario_bdd:
    nome: CNPJ inválido durante edição
    tipo: negativo
    gherkin: "Feature: Edição de banco existente\n  Scenario: CNPJ inválido durante edição\n    Given que existe um banco cadastrado com código 'BAN004'\n    When abro o registro 'BAN004' para edição\n    And altero o campo 'CNPJ' para '98.765.432/0001-11'\n    When clico no botão 'Salvar'\n    Then a mensagem 'CNPJ inválido ou não encontrado' aparece\n    And o registro não é atualizado"

cenario_bdd:
    nome: Salvar sem fazer nenhuma alteração
    tipo: positivo
    gherkin: "Feature: Edição de banco existente\n  Scenario: Salvar sem fazer nenhuma alteração\n    Given que existe um banco cadastrado com código 'BAN005'\n    When abro o registro 'BAN005' para edição\n    And clico no botão 'Salvar' sem mudar nenhum campo\n    Then a mensagem 'Nenhuma alteração detectada' é exibida\n    And o registro permanece inalterado"

cenario_bdd:
    nome: Campos obrigatórios vazios após edição
    tipo: negativo
```
