**US01 – Cadastro de Usuário**  
```yaml
titulo: US01 – Cadastro de Usuário,
cenario_bdd:
  - nome: Cadastro com dados válidos
    tipo: positivo
    gherkin: |
      Feature: Cadastro de Usuário
      Scenario: Cadastro com dados válidos
        Given o usuário abre a página de cadastro
        When ele preenche todos os campos obrigatórios com dados válidos
        And ele clica em "Cadastrar"
        Then uma mensagem de confirmação "Cadastro concluído" é exibida
        And o usuário pode fazer login imediatamente
  - nome: Cadastro bloqueado quando campo obrigatório está vazio
    tipo: negativo
    gherkin: |
      Feature: Cadastro de Usuário
      Scenario: Cadastro bloqueado por campo obrigatório vazio
        Given o usuário abre a página de cadastro
        When ele deixa o campo "CPF" vazio e tenta cadastrar
        Then o botão "Cadastrar" permanece desabilitado
        And a mensagem de erro "Campo obrigatório" aparece ao lado do campo
  - nome: Cadastro rejeitado com telefone inválido
    tipo: negativo
    gherkin: |
      Feature: Cadastro de Usuário
      Scenario: Cadastro rejeitado com telefone inválido
        Given o usuário abre a página de cadastro
        When ele preenche o campo "Telefone" com "12345"
        And tenta cadastrar
        Then a mensagem de erro "Telefone inválido" é exibida
        And o cadastro não é processado
  - nome: Cadastro rejeitado com CEP em formato errado
    tipo: negativo
    gherkin: |
      Feature: Cadastro de Usuário
      Scenario: Cadastro rejeitado com CEP em formato errado
        Given o usuário abre a página de cadastro
        When ele preenche o campo "CEP" com "1234-567"
        And tenta cadastrar
        Then a mensagem de erro "CEP inválido" é exibida
  - nome: Cadastro rejeitado com e‑mail sem “@”
    tipo: negativo
    gherkin: |
      Feature: Cadastro de Usuário
      Scenario: Cadastro rejeitado com e‑mail sem “@”
        Given o usuário abre a página de cadastro
        When ele preenche o campo "E‑mail" com "usuario.com"
        And tenta cadastrar
        Then a mensagem de erro "Formato de e‑mail inválido" é exibida
```
